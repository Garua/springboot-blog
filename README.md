# springboot-blog
**一个功能基本齐全的个人博客 **

##### *前端页面基于b站上某位大佬的视频了一下*。

#### 项目主要技术：

- Spring-Data-JPA
- Shiro
- Elasticsearch
- MySQL
- Semantic-ui

#### 主要功能

> 后台管理

> > 博客管理 :草稿，发布，删除，增加。
> >
> > 分类管理:单个或者批量插入，用,分隔（自动去重）
> >
> > 标签管理:单个或者批量插入，用,分隔（自动去重）
> >
> > 友链管理:新增或者删除友链
> >
> > 用户管理:禁言或者修改用户角色。--

> 前端用户

> > 登录，注册，接入qq登录，绑定或者解绑qq
> >
> > 评论以及回复评论
> >
> > 搜索博客并高亮显示（由于刚学es，技术原因，所以搜索这里只能搜索汉字）
